# SAM Simulator-2

Pseudo-simulator of the surface-to-air system, based on Deno, Vue 3, Canvas&Konva.js

## Used technologies
Client:
- Vue 3
- Konva.js (VueKonva)
- Typescript
- Vuetify
- Pinia

Server:
- Deno
